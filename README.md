# YGOPRO Blogger Tool MVP

這個 MVP 改成「文字編輯為主、程式自動排版」的流程，不需要讓一般使用者直接修改 HTML。

## 目前功能

- 載入 `cards.cdb`，搜尋卡名或卡號
- 讀取 `pics/` 內以卡號命名的圖片
- 用 `+ 新增段落` 自由建立段落與段落名稱
- 在目前焦點段落一鍵插入標準卡片段落
- 卡片段落自動帶入：
  - 卡名
  - 效果文
  - 圖片
  - 作者評論
- 編輯時右側即時預覽 HTML 成品
- 查詢卡片時顯示卡圖預覽
- 資料來源區可隱藏 / 顯示
- 右側即時預覽 Blogger HTML 成品
- 一鍵複製匯出 HTML
- 一鍵儲存匯出 HTML

## 啟動方式

```powershell
.\venv\Scripts\python.exe .\main.py
```

## 使用方式

1. 啟動程式後，確認上方路徑：
   - `cards.cdb`
   - `pics`
   - 輸出 HTML 位置
2. 在左側搜尋並選擇卡片。
3. 如已有 Blogger 或圖床圖片網址，可填入 `公開圖片 URL`。
4. 右側按 `+ 新增段落` 建立文章段落。
5. 每個段落可以自行決定是否填寫段落名稱。
6. 把游標放到想插卡的那個段落文字區。
7. 在左側 `作者評論` 輸入這張卡的評語。
8. 按 `插入卡片到目前段落`。
9. 右側會即時預覽最終 HTML 成品。
10. 完成後按 `複製匯出 HTML` 或 `儲存匯出 HTML`。

## 卡片插入格式

正文編輯區內會插入可讀的文字區塊，匯出時會自動轉成標準 Blogger HTML：

```text
【卡片】
卡名：卡片名稱
圖片：圖片網址或本機預覽路徑
效果：
這裡是效果文
評論：
這裡是作者評論
【/卡片】
```

## 打包

```powershell
.\build.ps1
```

輸出檔會在 `dist/YGOPRO_Blogger_Tool.exe`。

## 目前限制

- 預覽時會優先使用本機圖片。
- 匯出時若圖片來源是本機檔案，會自動轉成 `{{IMAGE_URL:卡號}}` placeholder。
- 目前版型是程式內建的標準版，還沒有做多模板切換。
