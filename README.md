# YGOPRO Blogger Tool

這是一個用來整理遊戲王卡片介紹文章的桌面工具，重點是讓使用者不用自己手寫 HTML，也能快速編輯內容、查卡、預覽卡圖，並整理出適合後續貼到 Blogger 的文章內容。

## 功能介紹

- 搜尋遊戲王卡片
- 顯示卡名、圖片編號、卡圖預覽、效果文
- 將卡片內容複製成適合貼到 Word 的圖片與文字格式
- 直接編輯文章內容
- 貼上既有 HTML 草稿後再整理
- 複製匯出 HTML

## 使用方式

1. 下載並啟動 `YGOPRO_Blogger_Tool.exe`
2. 開啟程式後，先到上方 `資料來源`
3. 手動選擇你自己的：
   - `cards.cdb`
   - `pics/` 資料夾
4. 開始搜尋卡片與編輯文章

## 重要提醒

- 打包檔 **不會內含** `cards.cdb`
- 打包檔 **不會內含** `pics/` 資料夾
- 你必須在自己的電腦上準備好 `cards.cdb` 與 `pics/`，並在程式內手動選擇
- 如果沒有先選資料來源，程式不會顯示卡片資料

## 適合的使用情境

- 撰寫遊戲王主題介紹文
- 整理單卡介紹
- 先在 Word 類編輯器排版，再轉貼 Blogger
- 將舊 HTML 草稿重新整理成較乾淨的內容
